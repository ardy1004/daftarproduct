📌 Spesifikasi Lengkap Website E-Commerce "Daftar Product"
1. Desain & Layout

Warna utama: kombinasi hijau zamrud, biru metalik, violet, kuning.

Kriteria: responsif, user-friendly, mobile-friendly, modern futuristik.

Frontend: Next.js + Tailwind CSS.

2. Halaman Utama

Header: search produk berdasarkan keyword.

Banner Carousel:

Menampilkan produk unggulan yang dipilih manual oleh admin via dashboard.

Produk unggulan bisa lebih dari 1 → tampil otomatis auto-slide.

Query khusus:

select * from products 
where is_featured = true 
order by featured_order asc;


Section Produk Terbaru: tampilkan maksimal 4 produk terbaru.

Produk Grid/List dengan Filter:

Filter kategori & subkategori (checkbox dinamis dari database).

Filter harga (slider + input manual rentang harga).

Sort by: Populer (default, klik terbanyak), Terlaris (sales terbanyak), Harga termurah, Harga tertinggi, Rekomendasi.

Tampilan produk (ProductCard):

Overlay label: “Terlaris” / “Rekomendasi”.

Image produk, Product ID, judul, rating (bintang) + jumlah penjualan.

Harga.

Tombol "Lihat Produk" → link ke affiliate_url (tanpa menampilkan kata "affiliate").

Interaksi Tombol "Lihat Produk"

Cursor berubah jadi pointer.

Hover effect: perubahan warna background (shade lebih gelap/terang).

Animasi transisi halus (0.2s) + efek scaling kecil (scale-105).

Tampilan Produk Mobile

Grid produk mobile = 2 kolom sejajar horizontal, mirip Shopee.

Grid produk desktop/tablet lebih banyak kolom.

Implementasi Tailwind:

Pengaturan Banner Carousel

Kolom tambahan pada produk:

alter table products
add column is_featured boolean default false,
add column featured_order integer;


Admin bisa centang toggle "Tampilkan di Banner" pada produk.

Bisa atur urutan slide dengan drag & drop.

Pengaturan Filter

Toggle ON/OFF untuk menampilkan/menyembunyikan Filter Kategori & Subkategori di frontend.

Disimpan di tabel settings:

create table if not exists settings (
  id uuid default gen_random_uuid() primary key,
  show_category_filter boolean default true,
  updated_at timestamp with time zone default now()
);

5. Fitur Filter Produk
🔹 Filter Harga (Slider + Input Manual)

Slider rentang harga + input manual (Harga Minimum / Maksimum).

Sinkron otomatis slider ↔ input.

Validasi:

Jika min > max → swap otomatis.

Input angka saja (0 – ∞).

Auto-format ribuan (contoh: 1.000.000).

Query:

select * from products where price between min_price and max_price;

🔹 Filter Kategori & Subkategori

Checkbox kategori (diambil dinamis dari Supabase).

Jika kategori dicentang → tampil semua produk kategori itu.

Jika subkategori dicentang → tampil produk sesuai kategori + subkategori.

Bisa multi-select.

Query:

select * from products 
where category in (...) 
  and subcategory in (...);


Filter hanya muncul jika setting show_category_filter = true.
